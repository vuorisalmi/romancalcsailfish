Roman Calculator for the SailfishOS (Jolla)

The Roman Calculator is my personal application porting test/example
for different platforms. It was originally written in Python/Qt for
the Nokia Maemo platform and later re-written on QML/JavaScript for
MeeGo/N9. That was easily ported to SailfishOS.
